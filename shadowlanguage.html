Shadow Language Syntax

To use the Shadow Language, provide values in this exact order, separated by pipes (|):

  • 1st: Graphics Mode (NASA PC, mid rig, potato)
  • 2nd: Distance (any value from -485 to 485)
  • 3rd: Smooth Graphics (0 or 1)
  • 4th: Light Emittance (any value from 30 to 200)
  • 5th: Floor (1 or 0)

Example: NASA PC|485|1|85|0|
